Class SoundPlayer

Package

(default package)

Inheritance

java.lang.Object → SoundPlayer

Description

Handles the playback of sound effects or background music using the Java Sound API. Supports playing audio clips once or looping continuously.

Constructor Summary

Constructor Description
SoundPlayer() Creates a new instance of SoundPlayer without any loaded sound clip.

Method Summary

Modifier and Type Method Description
public void playSound(String soundFile, boolean loop) Plays the specified audio file. Optionally loops the clip continuously if loop is true. Reduces volume by 15dB.
public void stop() Stops playback if a clip is currently running.
