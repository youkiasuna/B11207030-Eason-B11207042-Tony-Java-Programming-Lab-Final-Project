Class GameManager

Package

(default package)

Inheritance

java.lang.Object → GameManager

Description

Manages the game state and transitions between the main menu and game panels. Handles level progression and completion feedback for the Gold Miner game.

Constructor Summary

Constructor Description
GameManager(MainFrame frame) Initializes the game manager with the main application frame and creates the menu panel.

Method Summary

Modifier and Type Method Description
public void startGame(int level) Starts the game at the specified level and switches the content pane to the game panel.
public void startNextLevel(int currentLevel) Proceeds to the next level, or shows a completion message if all levels are cleared.
public void returnToMenu() Returns the view to the main menu panel.
public MenuPanel getMenuPanel() Returns the menu panel instance.
public GamePanel getGamePanel() Returns the game panel instance.
