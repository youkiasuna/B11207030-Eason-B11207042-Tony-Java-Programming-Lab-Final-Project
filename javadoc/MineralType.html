🪨 MineralType 枚舉類別

MineralType 定義了遊戲中支援的礦物種類，並與各自的圖像檔案建立關聯。

📦 Enum 成員

  • GOLD：金塊，使用圖檔 gold.png
  • ROCK：石頭，使用圖檔 rock.png

🧰 欄位與方法

private final String imageName;

對應圖片的檔案名稱，儲存在 /images 資料夾下。

MineralType(String imageName)

建構子，設定該礦物類型對應的圖片檔名。

public Image getImage()

呼叫 ImageLoader.loadImage() 載入並回傳對應圖像物件。

public String getImageName()

回傳礦物圖像的檔名。

📋 用法範例


MineralType type = MineralType.GOLD;
Image img = type.getImage();

📎 備註

  • 此 enum 通常由 Mineral 類別中的 type 欄位使用。
  • 擴充新礦物類型時，只需新增 enum 值與對應圖檔即可。
