Class GamePanel

```

Package

(default package)

Inheritance

java.lang.Object → java.awt.Component → java.awt.Container → javax.swing.JComponent → javax.swing.JPanel → GamePanel

Description

Represents the main gameplay panel for the Gold Miner game. Handles rendering, game updates, input events, and win/loss conditions.

Constructor Summary

Constructor Description
GamePanel(GameManager manager, int level) Initializes the panel with game resources, assets, and state for the given level.

Method Summary

Modifier and Type Method Description
public void actionPerformed(ActionEvent e) Main game loop logic executed by timer. Updates hook, game objects, and checks win/loss conditions.
protected void paintComponent(Graphics g) Renders the game scene, including background, miner, minerals, mice, and HUD.

Key Events

  • SPACE: Launches the hook to grab items.
  • C: Uses a bomb to destroy the currently caught item if possible.

Fields

  • ArrayList<Mineral> minerals - List of collectible resources.
  • ArrayList<Mouse> mice - List of moving mice in the level.
  • ScoreManager scoreManager - Tracks current and target score.
  • TimerManager timerManager - Handles time countdown.
  • Hook hook - The player's grappling hook.
  • SoundPlayer bgmPlayer, sfxPlayer - Background music and sound effects manager.
  • int level - Current game level.
  • int bombsLeft - Remaining bombs.
  • boolean gameEnded - Whether the game has finished.

UI Elements and Assets

  • Background images: background_ground.jpg, background_underground.jpg
  • Miner sprite: miner.png
  • Music: bgm.wav
  • Sound Effects: explosion.wav
```
