🗺️ LevelLoader 類別

LevelLoader 是用來載入不同遊戲關卡資料的靜態類別。每一關的礦物、老鼠、目標分數、時間與炸彈數都在這裡預先配置。

📦 屬性

  • public static final int MAX_LEVEL = 9：定義遊戲最大關卡數。

🧰 方法

public static LevelConfig loadLevel(int level)

根據關卡編號（1-9）回傳對應的 LevelConfig 設定物件。

參數

  • level：關卡編號，從 1 到 MAX_LEVEL。

回傳值

對應的 LevelConfig 物件；若編號無效，預設回傳第一關設定。

📋 關卡內容範例

關卡 1（新手教學）

  • 黃金與岩石共 3 個
  • 普通老鼠與鑽石老鼠各 1 隻
  • 目標分數：400，時間限制：60 秒，炸彈：3 顆

關卡 5（炸彈教學）

  • 黃金被高硬度岩石包圍，必須使用炸彈
  • 目標分數：1400，時間限制：30 秒，炸彈：4 顆

關卡 9（終極地獄）

  • 鋪滿超高重量岩石 + 超高速鑽石老鼠
  • 時間壓力大，目標分數高達 4200，炸彈數：0

📎 備註

  • 此類別內建所有關卡內容，因此維護或新增關卡需修改原始碼。
  • 通常由 GameController 或 GameScene 呼叫。
  • 每一個 Mineral 與 Mouse 都根據其座標、價值、重量與圖片尺寸設定。

📘 使用方式


LevelConfig level3 = LevelLoader.loadLevel(3);
int time = level3.getTimeLimit(); // 40
int bombs = level3.bombCount;     // 4
List<Mineral> minerals = level3.minerals;
