Class MainFrame

Package

(default package)

Inheritance

java.lang.Object → java.awt.Component → java.awt.Container → javax.swing.JComponent → javax.swing.RootPaneContainer → javax.swing.JFrame → MainFrame

Description

Main application window for the Gold Miner game. Sets up the game window, initializes the game manager, and displays the main menu panel.

Constructor Summary

Constructor Description
MainFrame() Creates the main application frame with a title, default size, and menu panel as the content pane.

Method Summary

Modifier and Type Method Description
public static void main(String[] args) Launches the game using SwingUtilities.invokeLater. Also prints the working directory for debugging purposes.

Fields and UI Configuration

  • Title: "Gold Miner Game"
  • Window Size: 800 x 600 pixels
  • Default Close Operation: EXIT_ON_CLOSE
  • Centers the window on screen using setLocationRelativeTo(null)
  • Initial panel: MenuPanel from GameManager
