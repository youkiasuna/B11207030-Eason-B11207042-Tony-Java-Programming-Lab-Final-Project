🖼️ ImageLoader 類別

此工具類負責從專案的 /images/ 資料夾中載入圖片，支援 Swing 圖像的標準用法。

🔧 方法說明

public static Image loadImage(String fileName)

嘗試載入指定檔名的圖片。如果成功，回傳 Image 物件；若失敗，會印出錯誤訊息並回傳 1x1 的透明圖像（預防 null pointer 錯誤）。

📌 使用範例

Image img = ImageLoader.loadImage("hook.png");

📁 圖片檔案應放置路徑

所有圖片需放在 resources/images/ 目錄下，例如：

  • /resources/images/hook.png
  • /resources/images/miner.png
  • /resources/images/background_underground.jpg

⚠️ 錯誤處理

若載入失敗，將顯示錯誤於標準輸出：

Failed to load image: hook.png - Invalid image or not found

📋 備註

  • 此類別依賴 getResource()，因此需確保資源打包於 JAR 時仍可被存取。
  • 對無效圖片自動提供預設透明圖像，可防止繪製時噴錯。
