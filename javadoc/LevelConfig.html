🎮 LevelConfig 類別

LevelConfig 是遊戲中用來設定每個關卡的參數資料類別，包含礦物、老鼠、分數目標、時間限制與炸彈數量。

📦 欄位說明

  • ArrayList<Mineral> minerals：本關卡中出現的所有礦物清單。
  • ArrayList<Mouse> mice：本關卡中出現的所有老鼠清單。
  • int targetScore：該關卡的過關目標分數。
  • int timeLimit：關卡限制時間（單位：秒）。
  • int bombCount：玩家於該關卡可使用的炸彈數量。

🧰 建構子


public LevelConfig(ArrayList<Mineral> minerals, ArrayList<Mouse> mice, int targetScore, int timeLimit, int bombCount)

用於初始化關卡設定的建構子。

參數

  • minerals：礦物物件清單
  • mice：老鼠物件清單
  • targetScore：過關分數門檻
  • timeLimit：倒數計時秒數
  • bombCount：可用炸彈數量

📘 方法

public int getTimeLimit()

取得此關卡的時間限制（秒）。

📋 使用範例


ArrayList<Mineral> minerals = new ArrayList<>();
minerals.add(new Mineral(...));

ArrayList<Mouse> mice = new ArrayList<>();
mice.add(new Mouse(...));

LevelConfig level = new LevelConfig(minerals, mice, 1000, 60, 2);

int time = level.getTimeLimit(); // 60

📎 備註

  • 本類別僅作為資料容器，不含邏輯運算。
  • 通常由 GameLevel 或主遊戲控制器使用。
