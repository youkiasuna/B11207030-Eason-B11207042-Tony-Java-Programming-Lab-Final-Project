Class GamePanel

```

Package

(default package)

Inheritance

java.lang.Object → java.awt.Component → java.awt.Container → javax.swing.JComponent → javax.swing.JPanel → GamePanel

Description

The main game panel responsible for rendering game graphics, handling game logic and user input for each level in the Gold Miner game.

Constructor Summary

Constructor Description
GamePanel(GameManager manager, int level) Initializes the panel, sets up game objects, starts the timer and loads background images for the specified level.

Method Summary

Modifier and Type Method Description
public void actionPerformed(ActionEvent e) Invoked by the timer; updates game state, checks win/loss conditions, and triggers scene changes.
protected void paintComponent(Graphics g) Draws the game elements, including background, miner, hook, minerals, and game statistics.

Fields and Game State

  • javax.swing.Timer timer – Controls frame updates.
  • Hook hook – Represents the hook mechanism for catching items.
  • ArrayList<Mineral> minerals – List of collectible mineral objects.
  • ArrayList<Mouse> mice – List of mice objects.
  • ScoreManager scoreManager – Tracks and manages score state.
  • TimerManager timerManager – Manages countdown timer.
  • Image backgroundGround, backgroundUnderground, minerImage – Visual elements.
  • int level – Current level number.
  • int bombsLeft – Number of bombs available to the player.
  • boolean gameEnded – Flag to indicate game has finished.
  • GameManager gameManager – Manages game state transitions.
  • SoundPlayer bgmPlayer, sfxPlayer – Background music and sound effect handlers.

Key Bindings

  • SPACE – Launches the hook to grab items.
  • C – Uses a bomb if one is available and an item is caught.
```
