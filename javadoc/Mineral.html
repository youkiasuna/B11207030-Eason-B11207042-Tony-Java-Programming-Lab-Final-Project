💎 Mineral 類別

此類別代表遊戲中的一個「礦物」（例如：金塊、石頭、鑽石等），具有位置、大小、價值、重量與圖像等資訊，並支援旋轉動畫及繪圖。

📦 屬性

  • x, y：礦物的中心位置座標。
  • value：礦物的得分價值。
  • weight：影響鉤爪拉回速度的權重。
  • type：礦物的種類（透過 MineralType 提供圖像）。
  • isCollected：是否已被鉤爪抓取。
  • isDestroyed：是否已被炸彈破壞。
  • rotation：當前旋轉角度。
  • rotationSpeed：旋轉速度（每次更新增加角度）。

🔧 方法

public void update()

更新旋轉角度，若礦物已被收集或破壞則不更新。

public void updateRotation()

與 update() 類似，但是另一組旋轉邏輯（如 hook 上的跟隨轉動）。

public void draw(Graphics g)

在遊戲畫面上繪出礦物，若有對應圖像會旋轉後繪製；否則畫一個黃色圓形。

public void drawAt(Graphics g, int x, int y)

在指定座標繪製礦物（用於被鉤爪抓住時顯示）。

public Rectangle getBounds()

取得礦物的邊界方形區域（用於碰撞檢查）。

public void setCollected(boolean) / isCollected()

標記礦物是否已被抓取。

public void setDestroyed(boolean) / isDestroyed()

標記礦物是否被炸彈破壞。

public int getValue()

取得礦物的分數。

public int getWeight()

取得礦物的重量（影響鉤爪返回速度）。

📋 備註

  • 需配合 MineralType 使用，以取得每種礦物的圖片資源。
  • 會自動根據 rotationSpeed 增加旋轉角度。
  • 可以被鉤爪抓取、繪製、炸彈銷毀。
